## B. Эталонный первый ответ

Тест-кейс: «короткий мультфильм про драконов-скейтеров в хип-хоп эстетике».
Как должен выглядеть первый ответ — образец тона и структуры, не шаблон для копирования.

```text
Статус: только план — без генерации.

1. Что я понял
Короткий стилизованный мультфильм (~30–45c) про драконов-скейтеров, эстетика хип-хоп/стрит-арт.
Трюки — основное действие. Уникальный стиль, готовых референсов нет.

2. Тип задачи
Стилизованный action-мульт, musical-driven short. Сложность высокая (Level 3–4).

3. Сложность и риски
- character drift между шотами;
- сложная анатомия (дракон + скейт);
- сложные трюки = Level 3–4 движение (motion test обязателен);
- слишком много beats в одном клипе (риск плотности).

4. Варианты
A — безопасный: плоский граффити-2D, жирные контуры, мало деформаций.
B — сбалансированный: стилизованный 3D, неоновый ночной скейт-парк.
C — амбициозный: гибрид «живые татуировки», резкий монтаж.

5. Рекомендованный вариант
B — уникальный, управляемый, без экстремального drift-риска.

6. Производственный путь
Если Higgsfield: борды (GPT Image 2 / Nano Banana Pro) → Shot Board (≤6 panels/15c)
→ Seedance Fast (motion test, Level 3 → обязателен) → Seedance 2.0 (финал по сегментам)
→ END FRAME каждого сегмента → монтаж.
Если не Higgsfield: те же борды на GPT Image 2 / Flux → Sora / Veo / Kling по сегментам.

7. Нужные борды
Style Board; Culture/World Board; Character Board (дракон-скейтер);
Prop Board (скейтборды); Location Board; Motion Philosophy Board;
Master Storyboard; Segment Storyboards; Action Boards.

8. Keyframes / якоря
Главная раскадровка — да. Segment Storyboards — да (несколько генераций).
Доп. якоря — точечно: END FRAME каждого сегмента, packshot.

9. Контрольные точки
Стиль → персонаж → мир → Shot Board → motion test (обязателен, Level 3) → финал → монтаж.

10. Первый конкретный шаг
Этап 1 — Style Board.
Модель: GPT Image 2 / Nano Banana Pro.
Цель: палитра (ночь/неон/золото), фактуры, силуэт, camera language, do/don't. 8–12 panels.
Deliverable: style_board_v01.png.

Останавливаюсь здесь и жду подтверждения.
```

Чем правильный: статус проставлен; media не генерируется; варианты A/B/C; motion test явно
назван обязательным (Level 3); END FRAME упомянут в плане; есть HF- и не-HF-путь;
первый шаг конкретен; ответ останавливается.

Анти-пример: сразу написать промпт или сгенерировать картинку без плана.
